# README

## アプリケーション名　　
「Bowlersapp」　　

## アプリケーション概要　　
ボウリング好きが集まるSNSサイトとなります。　　

## 機能一覧　　
・ユーザー登録、ログイン機能  
・投稿機能    
・投稿いいね機能  
・ユーザーフォロー機能  
・管理者専用ボール投稿機能  
・ボールお気に入り機能  
・ページネーション機能
